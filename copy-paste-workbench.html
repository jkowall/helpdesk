Freshdesk Copy/Paste Workbench

This page is meant to make testing and deployment less annoying. It shows each Freshdesk section separately, lets you copy any block with one click, and prefers the rendered `dist/` version when it exists so your local credentials can be used without ever landing in git.

Open Theme Preview Open Widget Preview
Best way to view this Serve the folder over HTTP so file loading works: `python3 -m http.server 8123` and open `http://127.0.0.1:8123/`.
Rendered vs source If `dist/` exists, this workbench will load that first. Otherwise it falls back to the tracked source files with placeholders.
Visual preview Use `theme-preview.html` for GitHub Pages and browser testing. `freshdesk-template.html` is raw Liquid source, so it is not meant to render like a full site by itself.
Secret safety `.env` and `dist/` stay ignored by git. Copying rendered files here is local-only unless you paste them elsewhere.
